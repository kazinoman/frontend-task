## Run Locally

Clone the project

```bash
   git clone https://github.com/kazinoman/frontend-task
```

Go to the project directory

```bash
  cd frontend-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
